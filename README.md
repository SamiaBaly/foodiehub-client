# 🍔 FoodieHub

A modern Full Stack Food Management Web Application built with **Next.js**, **Express.js**, **MongoDB**, **TypeScript**, and **Tailwind CSS**.

Users can browse foods, search by category, view food details, register/login, and manage food items based on their roles.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/SamiaBaly/foodiehub-client
```

```bash
git clone https://github.com/SamiaBaly/foodiehub-server
```

---

## Client Setup

```bash
cd food-client
npm install
```

Create a `.env.local`

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```

Run

```bash
npm run dev
```

---

## Server Setup

```bash
cd food-server
npm install
```

Create a `.env`

```env
PORT=5000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret

CLIENT_URL=http://localhost:3000
```

Run

```bash
npm run dev
```
